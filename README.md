# 🏏 IPL Analytics Dashboard | Power BI

## 📖 Overview

This Power BI dashboard provides comprehensive insights into Indian Premier League (IPL) statistics through interactive visualizations and performance metrics.

The dashboard enables users to analyze overall tournament records, batting performances, and bowling performances across IPL seasons using dynamic filters and KPI cards.

---

## 🎯 Objectives

- Analyze historical IPL data.
- Identify top-performing batters and bowlers.
- Track tournament records and milestones.
- Provide an interactive experience using Power BI slicers and visuals.

---

## 📊 Dashboard Pages

### 1️⃣ IPL Records

This page provides tournament-level insights including:

- Total Runs
- Total Wickets
- Total Matches
- Team Performance Analysis
- IPL Record Statistics
- Season-wise Trends

### 2️⃣ Batter Records

This page focuses on batting performance metrics:

- Most Runs
- Highest Individual Scores
- Strike Rate Analysis
- Batting Records
- Top Batters Comparison
- Boundary Statistics

### 3️⃣ Bowler Records

This page focuses on bowling performance metrics:

- Most Wickets
- Best Bowling Figures
- Bowling Records
- Top Bowlers Analysis
- Wicket Distribution
- Bowling Performance Comparison
